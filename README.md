## Overview

Zero Space is a personal cloud music platform that allows users to upload, store, and stream their own music library from anywhere.

Unlike traditional streaming platforms, Zero Space focuses on **user-owned audio libraries** and enables **phone-free listening through wearable devices**.

Users can upload rare tracks, DJ mixes, or personal recordings and stream them directly to Bluetooth headphones without needing a phone.

---

## Problem

Modern streaming platforms have several limitations:

• Rare or personal audio is not available on streaming services  
• Carrying a phone during workouts or running is inconvenient  
• Ads interrupt listening and focus  
• Devices cannot easily switch between listening contexts  
• Logging in repeatedly across devices wastes time  

---

## Solution

Zero Space introduces a **personal cloud audio system** where:

• Users upload any audio file format  
• Audio is automatically transcoded for streaming  
• Files are stored securely in cloud object storage  
• Wearable devices fetch streams directly via LTE or WiFi  
• Audio streams directly to Bluetooth earphones  

Result: **Phone-free music streaming with zero ads.**

## System Architecture

The Zero Space platform uses a cloud-native streaming architecture.

1. Upload Service  
Users upload audio via web or mobile apps.

2. Transcoding Queue  
Audio files are automatically converted into optimized streaming formats.

3. Cloud Storage  
Encrypted object storage stores user libraries.

4. Streaming Engine  
Adaptive bitrate streaming ensures reliable playback.

5. Wearable Gateway  
Smartwatch or wearable device fetches streams directly.

6. Bluetooth Playback  
Audio is transmitted to headphones using Bluetooth A2DP or BLE Audio.
